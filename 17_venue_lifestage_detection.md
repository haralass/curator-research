## Life-Stage Detection: 🟢 CONFIRMED NOVEL

No system detects structural life-stage transitions (student→professional, new project, new relationship) from file organizational patterns.

What exists but is different:
- Life event detection from **social media text** (CHI 2021) — text, user-initiated
- Life event detection from **wearable sensors** — physiological, acute events only
- PIM qualitative studies — observational, no automatic detection
- FileGram (2026) — workspace sync, not transition detection

**Framing:** "Prior work detects life events from social disclosure or physiological signals. We are the first to detect structural life-stage transitions from latent file organization patterns — ambient, continuous, requires no user action."
