# HiPEAC Vision 2011
**Σελίδες:** 56 | **Εποχή:** Πριν smartphones κυριαρχήσουν, Moore's Law ακόμα "δουλεύει"

---

## Κεντρικό Θέμα
Η απόδοση δεν μπορεί πια να προέλθει μόνο από συχνότητα (Dennard scaling τελειώνει). Χρειάζεται **παραλληλισμός** και **εξειδικευμένο hardware**.

## Βασικές Προβλέψεις / Recommendations
- Η ενέργεια (power) θα γίνει **ο κυριότερος περιορισμός** στο computing (σωστό)
- Χρειάζονται **πολυ-πύρηνα (many-core)** συστήματα — ο προγραμματισμός τους είναι open problem
- Embedded systems θα γίνουν κυρίαρχα (σωστό — IoT/mobile explosion)
- Ανάγκη για **cross-layer optimization**: hardware + compiler + OS + application μαζί

## Τι Δεν Είπε (αλλά ήρθε)
- Deep learning / neural networks: **απόντα εντελώς**
- Cloud computing: αναφορά αλλά όχι κεντρικό θέμα
- AI: περιθωριακό

## Σημασία για σένα
Baseline — δείχνει πόσο δραματικά άλλαξε το πεδίο σε 14 χρόνια. Ό,τι θεωρούνταν "μέλλον" τότε (many-core) είναι τώρα commodity.
