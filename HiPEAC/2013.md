# HiPEAC Vision 2013
**Σελίδες:** 48 | **Εποχή:** Smartphone boom, big data αρχίζει

---

## Κεντρικό Θέμα
**"Beyond Von Neumann"** — το παραδοσιακό μοντέλο CPU δεν είναι αρκετό πια. Χρειάζονται νέες αρχιτεκτονικές.

## Βασικές Προβλέψεις / Recommendations
- **Heterogeneous computing** (CPU + GPU + accelerators) είναι το μέλλον (σωστό)
- **Energy efficiency** ως πρώτης τάξης μετρική — όχι μόνο performance (σωστό)
- **Reconfigurable computing** (FPGAs) για adaptable hardware
- Ανάγκη για **programming models** που κρύβουν την πολυπλοκότητα από τον developer
- **Security** αρχίζει να εμφανίζεται ως concern

## Τι Δεν Είπε
- Machine learning ως dominant workload: **απόν**
- LLMs, transformers: **ανύπαρκτα** έννοια
- Edge AI: **δεν αναφέρεται**

## Σημασία για σένα
Η πρόβλεψη για heterogeneous computing ήταν 100% σωστή — αλλά κανένας δεν προέβλεψε ότι ο "killer app" για αυτό το hardware θα ήταν deep learning. Αυτό έγινε 1 χρόνο αργότερα (AlexNet 2012).
