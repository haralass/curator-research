# HiPEAC Vision 2015
**Σελίδες:** 72 | **Εποχή:** Deep learning αρχίζει να κυριαρχεί, IoT boom

---

## Κεντρικό Θέμα
**Silicon roadblocks** — το Moore's Law επιβραδύνεται σοβαρά. Η βιομηχανία πρέπει να βρει εναλλακτικές.

## Βασικές Προβλέψεις / Recommendations
- **Deep learning accelerators** αρχίζουν να αναφέρονται ως ανάγκη (πρώιμα σωστό)
- **IoT** = ο επόμενος μεγάλος χώρος — δισεκατομμύρια connected devices
- **Cyber-Physical Systems (CPS)**: η ένωση digital + physical world
- **Security/Privacy** ως fundamental challenge — όχι afterthought
- **Neuromorphic computing** αναφέρεται για πρώτη φορά ως ερευνητική κατεύθυνση
- **Cross-layer design**: από transistors ως applications

## Τι Άρχισε να Αναδύεται Σωστά
- Η ανάγκη για specialized hardware για ML workloads
- Ότι ο παραδοσιακός processor δεν φτάνει για AI

## Σημασία για σένα
Πρώτη αναφορά deep learning accelerators — δηλαδή αυτό που είναι σήμερα τα GPU/TPU/NPU. Αν κάποιος άρπαζε αυτή την κατεύθυνση τότε, θα ήταν στην αιχμή σήμερα.
