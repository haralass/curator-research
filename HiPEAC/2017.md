# HiPEAC Vision 2017
**Σελίδες:** 139 | **Εποχή:** Deep learning κυριαρχεί, autonomous vehicles hype, IoT εκρήγνυται

---

## Κεντρικό Θέμα
**Cognitive computing + CPS** — οι υπολογιστές αρχίζουν να "σκέφτονται" και να αλληλεπιδρούν με τον φυσικό κόσμο.

## Βασικές Προβλέψεις / Recommendations
- **Deep Learning accelerators** = επείγουσα ανάγκη (GPUs τότε, TPUs ξεκινούν)
  - *Σελ. 1177*: "CGRAs and neural network (Deep Learning) accelerators" — ακριβές
- **Autonomous vehicles** = killer app για embedded AI
- **Edge computing**: η επεξεργασία πρέπει να πλησιάσει τα δεδομένα
  - *Σελ. 1087*: "Cognitive systems will have an impact on hardware design (especially for embedded systems/edge computing)"
- **Security → Safety**: για CPS, ένα security bug σκοτώνει ανθρώπους
  - *Σελ. 1030*: "extension from security into safety and trustability"
- **Neuromorphic architectures** ως εναλλακτική για energy-efficient AI
  - *Σελ. 972*: αναφορά neuromorphic, Bayesian computing
- **Holistic design**: hardware + software + power + security μαζί
- **European sovereignty**: αρχίζει η ανησυχία για εξάρτηση από US/Asia

## Τι Προέβλεψε Σωστά
- Edge AI: ✅ (σήμερα = Apple Neural Engine, Qualcomm NPU, κλπ)
- Deep learning acceleration: ✅ (NVIDIA κυριαρχεί, Google TPU)
- Security as first-class citizen: ✅

## Τι Έχασε
- LLMs / transformers: 2017 είναι η χρονιά του "Attention is All You Need" — δεν το πρόβλεψε
- Scale of cloud AI: δεν φανταζόταν $200B data centers

## Σημασία για σένα
**Edge AI** είναι ο χώρος που συνδέεται άμεσα με το Curator (local inference). Η ανάγκη για "computation near the data" είναι ακριβώς το value proposition του τοπικού AI.
