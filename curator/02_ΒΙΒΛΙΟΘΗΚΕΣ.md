## ΜΕΡΟΣ 2: ΟΛΕΣ ΟΙ ΒΙΒΛΙΟΘΗΚΕΣ & ΕΡΓΑΛΕΙΑ

### Python Libraries — Clustering & Embeddings

| Βιβλιοθήκη | pip install | Χρήση |
|---|---|---|
| **river** | `pip install river` | DenStream online clustering (η κύρια βιβλιοθήκη για streaming) |
| **hdbscan** | `pip install hdbscan-with-cosine-distance` | Batch clustering αρχικού scan (σημ: αυτό το fork, όχι το vanilla hdbscan) |
| **BERTopic** | `pip install bertopic` | Topic modeling + merge_models για incremental |
| **sentence-transformers** | `pip install sentence-transformers` | SetFit classifier (μαθαίνει από user confirmations) |
| **setfit** | `pip install setfit` | Personal classifier από user history |
| **fastembed** | `pip install fastembed` | Γρήγορο ONNX embedding, 5000+ tokens/sec |
| **umap-learn** | `pip install umap-learn` | Dimensionality reduction πριν clustering |
| **scikit-learn** | (ήδη εγκατεστημένο) | Agglomerative clustering, silhouette scores |

### Near-Duplicate & Version Detection

| Βιβλιοθήκη | pip install | Χρήση |
|---|---|---|
| **unisim** | `pip install unisim` | RETSim (Google ICLR 2024) — version detection χωρίς filenames |
| **datasketch** | `pip install datasketch` | MinHash LSH για near-duplicate text (threshold 0.8 Jaccard) |
| **imagededup** | `pip install imagededup` | Image dedup: DHash (exact) + CNN MobileNet (near) |
| **imagehash** | `pip install imagehash` | pHash, dHash, wHash για images |
| **simhash** | `pip install simhash` | SimHash 64-bit fingerprints για text |
| **tree-sitter** | `pip install tree-sitter` | AST parsing για code version detection |
