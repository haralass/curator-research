## ΜΕΡΟΣ 6: Η ΑΡΧΙΤΕΚΤΟΝΙΚΗ ΠΟΥ ΠΡΟΤΕΙΝΩ

### Δύο Λειτουργίες

**Λειτουργία A — Deep Reorganize (μία φορά, τρέχει στα 5000 αρχεία)**

```
Φάση 1: Scan
  Walk ~/Downloads recursively → SQLite table
  Skip αρχεία με unchanged mtime+size (cache)
  ~45 λεπτά η πρώτη φορά

Φάση 2: Content Extraction (parallel, 4 threads)
  Images     → Moondream pre-screen + qwen2.5vl:7b
  PDFs       → Marker (ή MarkItDown για ταχύτητα)
  Office     → MarkItDown
  Code/Text  → plain read + tree-sitter AST

Φάση 3: Embedding
  Batch 20 texts/call → nomic-embed-text via Ollama
  OLLAMA_NUM_PARALLEL=4
  Store στο SQLite

Φάση 4: Duplicate Detection (παράλληλα)
  Exact:       SHA-256 hash
  Near-text:   MinHash LSH (datasketch, threshold 0.8)
  Near-image:  DHash≤2 (exact) + CNN 0.5 (near) via imagededup
  Versions:    RETSim cosine >0.86 (unisim)
  Code:        TSED via tree-sitter + apted

Φάση 5: NER + Temporal Co-occurrence
  spacy → extract entities per file
  Build co-occurrence matrix (shared entities = project signal)
  Build temporal graph (same modification sessions = project signal)

Φάση 6: Clustering
  HDBSCAN(min_cluster_size=5, metric='cosine') στα embeddings
  k-LLMmeans: LLM ονομάζει κάθε cluster (batch call)
  Agglomerative για sub-folders (2 levels)

Φάση 7: Misplacement Detection
  Per file: cosine_to_current vs cosine_to_best_other
  Delta > 0.15 → flagged as misplaced

Φάση 8: New Folder Detection
  GLOSH outlier scores → noise points
  Noise cluster size ≥ 3 → propose new folder
  Cosine to all centroids < 0.5 → new folder candidate

Φάση 9: Antilibrary + File Biography
  kMDItemUseCount == 0 + DateAdded > 6 months → archive candidates
  File value score (frequency + recency) → priority για approval UI

Φάση 10: JSON Manifest (ΤΙΠΟΤΑ ΔΕΝ ΕΧΕΙ ΚΙΝΗΘΕΙ ΑΚΟΜΑ)
  Κάθε proposed move: src, dst, confidence, reason, siblings preview

Φάση 11: Approval UI
  Group by destination
  Pre-select confidence > 0.9
  Show "potential siblings" (πιο πειστικό από score)
  Bridge nodes πρώτα για review
  20 items/page για 0.7-0.9 confidence

Φάση 12: Execute
  Dependency-ordered moves
  Leave symlinks για app reference preservation
  Full undo log (JSON)
```

**Λειτουργία B — Ongoing (νέα αρχεία καθημερινά)**

```
Νέο αρχείο στο inbox
  ↓
[macOS metadata] kMDItemWhereFroms, kMDItemCreator
  ↓ αν deterministic (π.χ. kMDItemCreator = Xcode) → sort αμέσως
  ↓
[Fast path] extension rules (.mp3, .zip, .dmg)
  ↓ αν match → sort αμέσως
  ↓
[Embedding] nomic-embed-text
  ↓
[DenStream query] cosine to existing micro-clusters
  ↓ αν cosine > 0.7 → route to that cluster's folder
  ↓ αν cosine 0.5-0.7 → route + flag for review
  ↓ αν cosine < 0.5 → new micro-cluster, wait for 3+ similar files
  ↓
[DenStream update] update micro-cluster centroid
  ↓
[Weekly] BERTopic merge_models για νέα topics
```
