## 1. ΤΙ ΕΙΝΑΙ ΤΟ CURATOR

Premium macOS εφαρμογή που **τακτοποιεί αυτόματα** τον φάκελο Downloads
με **τοπική AI** (Ollama). Τίποτα δεν φεύγει από τον Mac. 100% δωρεάν, 100% τοπικά.

**2 κομμάτια:**
- **Μηχανή** (`ai_classifier.py`) — τρέχει μόνη της όταν πέφτει αρχείο. Ταξινομεί.
- **Εφαρμογή** (`Curator.app`) — menu bar + παράθυρο UI για review/έλεγχο.

**Μοντέλα AI:** `llava:7b` (εικόνες), `llama3.2:3b` (κείμενο) — μέσω Ollama (localhost:11434).

## 8. CHANGELOG — 2026-05-20 (Βήμα 1: Ανοικτή AI Μηχανή)

### Πρόβλημα που λύθηκε
Το DSM-5-TR (διαγνωστικός οδηγός αυτισμού) πήγαινε λάθος στο EPL236 (Αλγόριθμοι).
Αιτία: ο ταξινομητής κοίταζε μόνο την **πρώτη σελίδα** του PDF σαν εικόνα και το LLM
έβλεπε λογική if/then → την ταύτιζε με αλγόριθμους. Επίσης, το σύστημα ήταν
«κλειστό» — υποχρεωτικά επέλεγε από σταθερή λίστα 15 κατηγοριών.

### Αρχιτεκτονική αλλαγή: ai_classifier.py

**Νέες βιβλιοθήκες (pip install στο Anaconda Python 3.13):**
- `markitdown` (Microsoft) — μετατρέπει PDF/DOCX/XLSX/PPTX/κώδικα σε markdown
- `ollama-instructor` — δομημένο Pydantic output από Ollama LLM
- `chromadb` — τοπική διανυσματική βάση (semantic similarity)
- `keybert`, `sentence-transformers` — αυτόματα tags από περιεχόμενο

**Δύο-φάσεις ταξινόμηση:**
1. **Εξαγωγή περιεχομένου**: markitdown για κείμενο, llava για εικόνες
2. **Συλλογιστική**: embedding → chromadb similarity → LLM με πλήρες context

**Ανοικτό σύστημα:**
- Το LLM αποφασίζει ελεύθερα (δεν του δίνουμε λίστα)
- Αν δεν υπάρχει κατάλληλος φάκελος → `suggest_new_folder` + αποθηκεύεται
  στο `~/Library/Scripts/curator_suggestions.json`
- Ο χρήστης ειδοποιείται να δημιουργήσει νέο φάκελο

**Honest uncertainty (καλύτερο να πει «δεν ξέρω»):**
- confidence ≥ 0.82 → HIGH → auto-sort
- 0.60–0.82 → MEDIUM → sort αλλά εμφανίζεται στο Recent
- < 0.60 → LOW → μένει στο Review, ο χρήστης αποφασίζει

**Νέα πεδία στο ClassificationResult (Pydantic):**
- `tags: List[str]` — αυτόματα tags (keybert)
- `rename_suggestion: Optional[str]` — πρόταση καλύτερου ονόματος
- `suggest_new_folder: Optional[str]` — πρόταση νέου φακέλου

**Persistence:**
- ChromaDB vectors: `~/Library/Scripts/curator_vectors/`
- Folder suggestions: `~/Library/Scripts/curator_suggestions.json`

### Αλλαγές Rust (lib.rs)
- `Entry` struct: προστέθηκαν `tags: Vec<String>`, `rename_suggestion: String`
- `read_undo_entries`: parse tags array από JSON
- Νέα struct `FolderSuggestion { filename, suggested_folder, time }`
- Νέες εντολές: `get_suggestions()`, `dismiss_suggestion(folder_name)`

### Αλλαγές TypeScript
- `curator.ts`: `Entry` interface + `tags`, `rename_suggestion`, `FolderSuggestion`
- `api.ts`: νέες functions `getSuggestions()`, `dismissSuggestion()`, humanized mock data

- Anaconda Python 3.13 (`/opt/anaconda3/bin/python3`) — εδώ εγκαθίστανται packages
- Το launchd χρησιμοποιεί το Anaconda Python
