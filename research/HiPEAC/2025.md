## ΚΕΦΑΛΑΙΟ 1: Next Computing Paradigm (NCP) — σελ. 24-30

### Digital Envelopes
Κάθε entity (άνθρωπος, αντικείμενο, εταιρεία) έχει:
1. **Intelligent digital agent** που δρα αυτόνομα για να εκπληρώσει goals
2. **Sensors** — διαβάζει inputs από τον κόσμο
3. **Actuators** — δρα στον κόσμο

Παράδειγμα (σελ. 28): disabled traveler — personal agent κλείνει ταξί, συντονίζει με σταθμό, πληρώνει τα πάντα αυτόματα.

### Edge-Cloud Continuum
- Computation γίνεται **εκεί που βολεύει**: local για privacy/latency, cloud για heavy lifting
- **XaaS (Anything-as-a-Service)**: το computing model του μέλλοντος
- Live migration: η εφαρμογή "κινείται" χωρίς interruption

### Key Recommendations (σελ. 13)
- Create **digital envelopes** standard
- Develop **AI-powered orchestrators** at the edge
- Expand **space- and time-aware protocols**
- Design **interoperable contract-based APIs**

## ΚΕΦΑΛΑΙΟ 2: Artificial Intelligence — σελ. 31-37

### Τι Έγινε το 2024
- **Smaller models = better**: GPT-3.5 (175B) → Gemma 2 (9B) με ίδια performance — 20x μείωση παραμέτρων
- **Inference compute scaling**: o1/o3 "σκέφτεται" πριν απαντήσει — variable compute time
  - *σελ. 34*: o3 = 97% στο AIME math competition
- **Multi-modal**: text + image + audio σε ένα model
- **Distributed training**: Prime Intellect — 10B parameter model trained across US/Europe/Asia
- **World models**: Oasis (Minecraft), DIAMOND (CS:GO) — AI simulates entire environments

### Recommendations
- **Specialized Action Models (SAMs)**: μικρά, εξειδικευμένα models ως services — σελ. 35
- **Orchestrators**: AI που επιλέγει ποιο SAM να χρησιμοποιήσει
- **Open protocols** για distributed agentic AI: "TCP/IP για AI agents" — σελ. 37
- **European foundation models + SAMs**: Mistral, Aleph Alpha ως παραδείγματα

### Open Research Problems (σελ. 35-37)
- **Protocol for AI-to-AI communication**: tokens; embeddings; natural language; — ανοιχτό
- **Orchestration algorithms**: πώς επιλέγεις τον καλύτερο SAM για κάθε task;
- **Non-functional requirements in AI APIs**: latency, hallucination rate, cost, privacy — πώς εκφράζεις αυτά;
- **Distributed training across jurisdictions**: legal + technical challenges

## ΚΕΦΑΛΑΙΟ 3: New Hardware — σελ. 38-41

### Trends
- **Inference chips** > training chips: o1/o3 model = inference-heavy
- **Near/In-Memory Computing (NMC/IMC)**: data + compute στο ίδιο chip — energy savings
- **Non-volatile memory**: direct execution χωρίς RAM load
- **Quantum**: Google Willow (105 qubits) — exponential error reduction — σελ. 39
  - Αλλά: practical applications ακόμα 10+ χρόνια
- **Heterogeneous integration**: διαφορετικά materials/technologies σε ένα chip

### Recommendations
- Develop inference engines for edge
- Invest in NMC/IMC research
- European sovereignty σε key hardware tech (photonics!)
- Modular + standardized hardware design

## ΚΕΦΑΛΑΙΟ 4: Tools (AI for SE) — σελ. 42-46

### Κατάσταση 2025
- AI tools generate "simple code of several pages" — σελ. 32
- Δεν αντικαθιστούν developers — αλλάζουν τι κάνουν οι developers

### Key Challenges (Research Gaps)
- *σελ. 44*: **Programming languages vs AI tools**: ποια γλώσσα πρέπει να target ένα AI coder;
- *σελ. 44*: **"Natural language → transistors"**: δεν είναι εφικτό ακόμα για large codebases
- *σελ. 45*: **Correctness + safety** = μεγαλύτερο barrier
  - Formal methods για AI-generated code: open problem
- *σελ. 45*: **Neural Architecture Search (NAS)**: automated design of neural networks
  - Multi-criteria optimization (power, memory, accuracy) — open
- European AI tools για sovereignty
